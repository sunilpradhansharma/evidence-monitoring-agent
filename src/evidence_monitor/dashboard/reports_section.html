{# How to read this page — definitions of every scale and status used below. #}
How to read this page
Sentiment scale
−1.0 (very negative about the therapy) to +1.0 (very positive); 0 is neutral. Favorable ≥ +0.30, concerning ≤ −0.30.
Competitive position
How the model ranks the therapy: first_line_recommended, among_options, second_line, not_recommended, not_mentioned.
Citation status
Whether the right indication was cited: cited, partial, absent, or wrong_indication (the model returned content for the wrong disease/indication).
Response status
success (complete answer), truncated (cut off at the token limit — partial text kept and still scored), failed (no answer after retries), blocked (provider safety filter).
